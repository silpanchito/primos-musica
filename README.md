# 🎵 Números Primos en la Música

---

## ¿Qué problema resuelve?

Analiza qué frecuencias (en Hz) dentro del rango audible humano 
son números primos, relacionándolas con notas musicales.

## Algoritmo utilizado

Se recorre el rango con un ciclo `for`. Para cada número se cuenta 
cuántos divisores tiene. Si el contador es exactamente 2, es primo.

## Tecnologías

- HTML5
- CSS3 (diseño responsivo)
- JavaScript (sin frameworks)

## Cómo usar

1. Ingresar una frecuencia inicial y una final en Hz
2. Presionar el botón "Encontrar primos en el rango"
3. Ver los resultados, estadísticas y tabla de notas musicales
